    Frequently Asked Queries

    Here are the most common questions about dedicated server hosting we get from our customers, Hence we elaborated our FAQ to ensure you find answers as quickly as possible.

    What is dedicated server hosting?

    A dedicated hosting service is a server that comes with dedicated features like RAM, storage, processing power, bandwidth, and much more. This means that these features will not be shared among others and are only entitled to that specific server. Dedicated server hosting is when a client rents this so-called dedicated server for a particular time period. This type of hosting is recommended because you will have full control over the server. To learn more about what is a dedicated server, you can check here.

    Does it matter what content I host on the server?

    1Gbits allows all kinds of content. As long as it doesnt violate the countrys laws, the website is hosted in or any laws currently active in Lithuania.

    How can I pay for the dedicated server?

    At 1Gbits, we accept multiple payment gateways to cater to the whole globe. We accept credit cards, bank transfers, PayPal, Bitcoin, Perfect Money, WebMoney, and all other cryptocurrencies. If you dont see your preferred payment method, contact us through the live chat, and we will provide it for you.
